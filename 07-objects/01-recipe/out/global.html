    recipes :Array.<Object>

    Array of recipes. Each recipe is represented as an object with the following properties: - `title`: The name of the recipe. - `serves`: The number of servings the recipe makes. - `ingredients`: An array of ingredients required for the recipe.

    serves number The number of servings the recipe makes.
